# SOSimulator — SOS团活动室

> 黑客松项目 | 凉宫春日主题卡牌模拟器

受《密教模拟器》(Cultist Simulator) 启发的网页卡牌桌面游戏。你扮演阿虚，在SOS团活动室中通过拖拽组合团员卡牌，探索世界的真相——同时要避免凉宫的情绪波动毁灭世界。

## 玩法

- **拖拽叠放**：将一张卡牌拖到另一张卡牌上，自动触发活动
- **配方系统**：特定组合触发特定事件（进度环+倒计时）
- **胜利条件**：收集"世界的真相"，交给阿虚 → 胜利
- **失败条件**：闭锁空间累积到 3 个 → 世界毁灭

### 关键配方

| 活动 | 组合 | 产出 |
|------|------|------|
| 日常搜索 | 凉宫 + 精力 | 日常线索 |
| 深入调查 | 阿虚 + 日常线索 | 超自然线索 |
| 情报解析 | 长门 + 超自然线索 | 世界的真相 |
| 未来预警 | 实玖瑠 + 精力 | 超自然线索 |
| 闭锁空间处理 | 古泉 + 闭锁空间 | 精力×2 |
| 安抚凉宫 | 凉宫 + 阿虚 | 消除闭锁空间 |
| 电影拍摄 | 凉宫 + 电影剧本 | 日常线索(不消耗剧本) |

## 角色

- **凉宫春日** — SOS团团长。对普通人类没有兴趣，却拥有改变世界的力量
- **阿虚** — 负责吐槽。唯一能安抚凉宫的人
- **长门有希** — 外星人。能解析一切数据
- **朝比奈实玖瑠** — 未来人。掌握关键时间情报
- **古泉一树** — 超能力者。「机关」成员，处理闭锁空间

## 开发

```bash
npm install
npm run dev      # 开发服务器 http://localhost:5173
npm run build    # 生产构建
npm run lint     # ESLint
```

## 技术栈

React 19 + TypeScript + Vite
