# 事件/剧情树文档

## 1. 剧情定位

剧情用于指导玩家理解卡牌组合意义，并推动从日常行动进入超自然行动。

## 2. 主线阶段

| 阶段 | 目标 | 关键产物 |
| --- | --- | --- |
| 日常 | 获得基础资源 | 金钱、精力 |
| 异兆 | 发现不寻常线索 | 奇怪线索 |
| 研究 | 将线索转化为知识 | 隐秘知识 |
| 梦境 | 获得仪式材料 | 梦境残响、意志 |
| 仪式 | 完成最终组合 | 门后之光 |
| 结局 | 触发胜利 | 窥见晨星 |

## 3. MVP 事件树

```text
start
  -> work_intro
  -> explore_hint

explore_hint
  条件：完成探索 + 贫瘠线索
  结果：获得奇怪线索
  -> research_unlock

research_unlock
  条件：拥有奇怪线索
  结果：解锁研究动作
  -> forbidden_note

forbidden_note
  条件：完成研究 + 奇怪线索
  结果：获得隐秘知识，记录剧情标记 saw_hidden_pattern
  -> dream_call

dream_call
  条件：拥有隐秘知识
  结果：解锁休息/梦境方向，获得梦境残响
  -> ritual_unlock

ritual_unlock
  条件：拥有隐秘知识 + 梦境残响
  结果：解锁仪式动作
  -> final_ritual

final_ritual
  条件：仪式输入 隐秘知识 + 梦境残响 + 意志
  结果：触发胜利事件
  -> ending_victory_morning_star
```

## 4. 支线占位

- 威胁线：调查者、债务、梦魇。
- 状态线：疲惫、惊惧、清醒、狂热。
- 道具线：旧地图、黄铜钥匙、无名手稿。

## 5. 结局骨架

### 胜利：窥见晨星

玩家完成仪式，看见门后的光。世界没有立刻改变，但玩家已经不再属于原来的日常。

### 失败：理智崩溃

玩家累积过多惊惧或禁忌状态，无法继续行动。

### 失败：归于日常

玩家长时间没有推进主线，线索消散。

MVP 只实现胜利结局，失败保留文档和数据占位。
