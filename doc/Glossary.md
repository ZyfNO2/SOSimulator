# 术语表

| 术语 | 英文/代码名 | 含义 |
| --- | --- | --- |
| 卡牌 | Card | 游戏中的资源、线索、状态、知识和结局载体。 |
| 卡牌定义 | CardDefinition | 描述一种卡牌的静态配置。 |
| 卡牌实例 | CardInstance | 桌面上一张具体卡牌。 |
| 动作框 | Action | 执行操作的容器，如探索、研究、仪式。 |
| 卡槽 | Slot | 动作框中的输入位置。 |
| 槽位需求 | SlotRequirement | 判断某张卡能否进入槽位的规则。 |
| 配方 | Recipe | 一组输入卡牌对应的动作结果。 |
| 标签 | Tag | 用于规则匹配的关键词。 |
| 倒计时 | Timer | 动作从开始到完成的时间。 |
| 产出 | Output | 动作完成后生成的卡牌、日志或事件。 |
| 消耗 | Consume | 动作完成后移除输入卡牌。 |
| 返还 | Return | 动作完成后输入卡牌回到桌面。 |
| 线索 | Clue | 推动探索与研究的卡牌类型。 |
| 知识 | Knowledge | 由研究产生的高阶卡牌类型。 |
| 仪式 | Ritual | 通向结局的高阶动作。 |
| 事件节点 | EventNode | 剧情树中的一个节点。 |
| 标记 | Flag | 记录剧情、解锁或状态的布尔/枚举数据。 |
| 日志 | Log | 玩家可回看的系统和剧情反馈。 |
| 结局 | Ending | 游戏结束状态，包含胜利或失败。 |
| MVP | Minimum Viable Prototype | 最小可玩原型。 |
