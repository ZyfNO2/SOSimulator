# 当前卡牌种类与产出链

## 1. 初始卡

开局只保留以下 5 类起点卡：

- `精力`
- `时间`
- `春日（平常）`
- `商店`
- `天气（晴）`

说明：

- `精力` 与 `时间` 仍然是左下角母牌，场上同类总量上限为 `2`。
- `春日（平常）/ 春日（筹备）/ 春日（执着）/ 春日（临界）` 表示同一角色在不同剧情推进阶段的状态卡。
- `蓝色雨伞（寻常）/（旧影）/（锚点）`、`失物招领室（寻常）/（留痕）`、`长门（观测）/（确认）` 也是同样的阶段式命名。
- `春日的执念 / 世界异常 / 不存在的失主 / 现实阻力 / 虚构失主` 这类名字不是人物，而是剧情关系、认知状态或异常状态。

## 2. 本轮产出规则

1. `商店 + 精力 -> 商店 + 金钱`
2. `金钱 + 商店 -> 商店 + SOS团活动材料`
3. `春日（平常） + SOS团活动材料 -> 春日（筹备）`
4. `春日（筹备） + 时间 + 精力 -> 春日（筹备） + 失物招领室（寻常）`
5. `失物招领室（寻常） + 精力 -> 失物招领室（寻常） + 蓝色雨伞（寻常）`
6. `失物招领室（寻常） + 蓝色雨伞（寻常） + 时间 -> 蓝色雨伞（寻常） + 失物招领室（留痕）`
7. `商店 + 蓝色雨伞（寻常） -> 蓝色雨伞（旧影） + 证词`
8. `春日（筹备） + 蓝色雨伞（旧影） -> 春日（筹备） + 蓝色雨伞（锚点） + 春日的执念`
9. `长门（观测） + 证词 -> 长门（确认）`
10. `春日（筹备） + 精力 + 春日的执念 -> 春日（执着）`
11. `蓝色雨伞（锚点） + 春日的执念 -> 春日的执念 + 世界异常 + 不存在的失主`
12. `世界异常 + 精力 -> 阿虚的怀疑`
13. `长门（确认） + 阿虚的怀疑 -> 结论`
14. `阿虚的怀疑 + 不存在的失主 -> 现实阻力`
15. `不存在的失主 + 春日的执念 -> 虚构失主`
16. 当前没有单独的第 16 条手动产出规则。
17. 当前不再通过手动产线推进天气阶段。
18. `春日（执着） + SOS团活动材料 -> 春日（临界）`

## 3. 自动触发规则

以下不是手动堆叠合成，而是“出现即触发”的剧情推进：

- `证词` 首次出现时，自动刷出 `长门（观测）`。
- 不再使用“`春日（执着）` 自动把 `蓝色雨伞（旧影）` 升级为 `蓝色雨伞（锚点）`”的旧逻辑，`蓝色雨伞（锚点）` 改为通过第 8 条配方产出。
- 首次发现任意阶段的 `蓝色雨伞` 后，天气开始进入独立计时。
- 天气不再随剧情进度手动推进，而是每 `3` 分钟自动上升一级：
  `天气（晴） -> 天气（阴） -> 天气（阵雨） -> 天气（大雨） -> 天气（暴雨） -> 天气（特大暴雨）`
- 天气卡上的倒计时遮罩与 `时间 / 精力` 母牌使用同一种表现。
- 当天气进入 `天气（特大暴雨）` 后，再过 `3` 分钟会强制结算 `虚构失主出现`。
- 强制结算时，场上其他所有牌都会清空，只留下 `虚构失主出现` 这一张结局牌。

补充：

- 所有 3 张及以上的规则里，第一个输入卡固定视为父牌。
- 从第二个输入开始，后面的输入顺序都可以互换。
- 也就是说第 6 条必须由 `失物招领室（寻常）` 当父牌，但 `蓝色雨伞（寻常）` 和 `时间` 在后两个位置里的先后可以互换。

## 4. 结局分歧

1. `天气（暴雨） + 世界异常 + 春日（临界） -> 灭世线`
2. `天气（大雨） + 结论 -> 阿虚认领蓝色雨伞`
3. `天气（大雨） + 虚构失主 -> 虚构失主出现`
4. 如果没有在更早的天气阶段完成其他结局，`天气（特大暴雨）` 倒计时结束后会强制进入 `虚构失主出现`

## 5. 命名约定

这版卡池统一按下面的方式理解：

- `名称（状态）`
  - 例如 `春日（平常）/（筹备）/（执着）/（临界）`、`长门（观测）/（确认）`
  - 含义是“同一个角色在不同剧情推进度下的状态”
- `线索 / 地点（状态）`
  - 例如 `蓝色雨伞（寻常）/（旧影）/（锚点）`、`失物招领室（寻常）/（留痕）`
  - 含义是“同一个线索或地点在不同调查深度下的版本”
- `关系 / 认知 / 异常名词`
  - 例如 `春日的执念`、`世界异常`、`不存在的失主`、`现实阻力`、`虚构失主`
  - 含义是“剧情中的抽象状态卡”，它们负责连接角色阶段、天气阶段和结局分支

也就是说，这些名字之间的联系可以统一理解为：

- 括号里的状态主要表示“阶段推进”
- 抽象名词主要表示“剧情状态”
- 天气卡主要表示“世界失稳程度”
- 结局卡主要表示“最后落点”
