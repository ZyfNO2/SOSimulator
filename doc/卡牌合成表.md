# 当前版卡牌合成表

### 精简版

1. 初始只有 `精力 / 时间 / 蓝伞 / 失物招领室事件卡`
2. `失物招领室 + 精力 -> 贫瘠线索`
3. `蓝伞 + 贫瘠线索 -> 模糊证词`
4. 首次获得 `模糊证词`，自动刷出 `春日` 和事件卡 `过于一致的证词`
5. `模糊证词 + 春日 -> 春日的执念`
6. `过于一致的证词 + 时间 -> 矛盾记录`
7. `过于一致的证词 + 金钱 -> 旧广播站照片`
8. `旧广播站照片 + 蓝伞 -> 雨中的背影`
9. `春日的执念 + 模糊证词 -> 世界异常`
10. 首次获得 `世界异常`，自动刷出 `阿虚` 和事件卡 `不存在的失主`###########
11. `阿虚 + 世界异常 -> 阿虚的怀疑`
12. `阿虚的怀疑 + 不存在的失主 -> 现实阻力`
13. `不存在的失主 + 春日的执念 -> 虚构失主`
14. `世界异常 + 雨中的背影 -> 最后一场雨`
15. 结局分歧：
16. `最后一场雨 + 阿虚的怀疑 -> 阿虚认领蓝伞`
17. `最后一场雨 + 虚构失主 -> 虚构失主出现`

这份表只描述新的文档目标玩法，不代表当前代码已经实现。

设计前提：

- 每日刷新 `精力 X2`
- 每日刷新 `时间 X2`
- 开局存在 `打工点`
- 第四天后 `打工点` 关闭
- 主线从 `失物招领室` 的蓝伞开始

---

## 1. 开局卡牌

建议开局桌面包含：

- `精力 X2`
- `时间 X2`
- `打工点`
- `开学日`
- `蓝伞`
- `失物招领室`

可选开局附加卡：

- `春日的指令`
- `SOS 团缺团费`

---

## 2. 日常线

## 2.1 基础打工链

| 输入 | 输出 | 说明 |
| --- | --- | --- |
| `打工点 + 时间` | `团费` | 最基础的挣钱行为 |
| `打工点 + 精力` | `团费` | 体力劳动版本 |
| `打工点 + 时间 + 精力` | `团费 + 打工见闻` | 更高投入，除了钱还能带出一点现实信息 |

## 2.2 店铺关闭后的变化

| 输入 | 输出 | 说明 |
| --- | --- | --- |
| `第 4 天` | `店铺关闭` | 事件触发卡 |
| `打工点 + 店铺关闭` | `关闭的卷帘门` | 打工点不再能提供资源转化 |

关闭后，以下配方应失效：

- `打工点 + 时间`
- `打工点 + 精力`
- `打工点 + 时间 + 精力`

---

## 3. 主线起步

## 3.1 看见蓝伞

| 输入 | 输出 | 说明 |
| --- | --- | --- |
| `开学日 + 失物招领室` | `路过失物招领室` | 进入事件入口 |
| `路过失物招领室 + 蓝伞` | `可疑的普通感` | 春日注意到蓝伞 |
| `可疑的普通感 + 春日` | `失物招领室事件卡` | 春日正式把它定义为事件 |

## 3.2 第一轮调查

| 输入 | 输出 | 说明 |
| --- | --- | --- |
| `失物招领室事件卡 + 时间` | `贫瘠线索` | 只是跑腿问人 |
| `失物招领室事件卡 + 精力` | `失物登记记录` | 做更实操的调查 |
| `贫瘠线索 + 蓝伞` | `模糊证词` | 开始有人说好像见过 |
| `失物登记记录 + 蓝伞` | `无人认领的异常` | “为什么一直没人来领”被突出出来 |

---

## 4. 主线中段

## 4.1 春日介入后

| 输入 | 输出 | 说明 |
| --- | --- | --- |
| `模糊证词 + 春日` | `春日的执念` | 春日把普通现象升级成故事 |
| `模糊证词 + 时间 + 精力` | `过于一致的证词` | 花更多行动获得更危险的线索 |
| `过于一致的证词 + 时间` | `矛盾记录` | 证词越来越多，但彼此开始打架 |
| `矛盾记录 + 精力` | `旧照片` | 拿到似是而非的物证 |

## 4.2 日常与异常开始争资源

| 输入 | 输出 | 说明 |
| --- | --- | --- |
| `打工见闻 + 模糊证词` | `现实说法` | 现实角度解释事件 |
| `春日的执念 + 旧照片` | `雨中的背影` | 春日把旧照片解释成命运痕迹 |
| `春日的执念 + 矛盾记录 + 时间` | `不存在的失主` | 失主不再像人，像一个被拼出来的概念 |

---

## 5. 异常升级

## 5.1 雨势推进

| 输入 | 输出 | 说明 |
| --- | --- | --- |
| `春日的执念 + 时间` | `小雨` | 主线开始影响天气氛围 |
| `小雨 + 模糊证词` | `中雨` | 异常被持续强化 |

| `大雨 + 雨中的背影` | `暴雨前夜` | 逼近最终段 |

| `现实阻力 + 时间` | `短暂清醒` | 给玩家一个喘气节点 |
| `春日的执念 + 不存在的失主` | `虚构失主` | 春日继续把故事推到现实中 |

## 7. 每日刷新建议

虽然当前代码还没做“按天刷新”，但文档层建议规则是：

| 日期 | 刷新内容 | 特殊说明 |
| --- | --- | --- |
| 第 1 天 | `精力 X2 + 时间 X2` | 打工和主线都开放 |
| 第 2 天 | `精力 X2 + 时间 X2` | 雨势可开始升级 |
| 第 3 天 | `精力 X2 + 时间 X2` | 主线消耗进一步变高 |
| 第 4 天 | `精力 X2 + 时间 X2` | 店铺关闭，打工线终止 |

---

## 8. 设计意图

这版合成表最关键的改变有三点：

1. 主线起步前先有日常线，故事不再直接从谜团开始。
2. 主线推进显著更吃 `时间` 和 `精力`，玩家会真实感觉到成本。
3. 第四天关店后，游戏会从“双线分配”转为“集中硬推主线”。

所以“把精力和时间全力推进主线”是可行的，但应该更紧、更险，而不是无代价的最优解。
